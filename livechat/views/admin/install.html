კონფიგურაცია

კეთილი იყოს თქვენი მობრძანება - ონლაინ დახმარება - ში-. დაწყებამდე, აპლიკაცია უნდა იყოს კონფიგურირებული, რომ მოხდეს მონაცემთა ბაზასთან დაკავშირება. ინსტალაციის დროს თქვენ დაგჭირდებათ შემდეგი ინფორმაცია (თუ არ იცი რომელიმე ფუნქციის შესრულება, შეგიძლიათ მიიღოთ ინფრომაცია თქვენი ვებ ჰოსტინგ პროვაიდერისგან):

1. Database ჰოსტი
2. Database პორტი (მხოლოდ იმ შემთხვევაში, თუ არის ნაგულისხმები)
3. Database სახელი
4. Database მოხმარებლები
5. Database პაროლი

თუ მოცემული მონაცემთა ბაზა ჯერ არ არსებობს, ინსტალაციის ოსტატი შეეცდება ავტომატურად შექმნას სერვერზე, თუმცა ეს მცდელობა ვერ მოხერხდება. არ ინერვიულოთ, უბრალოდ შექმენით მონაცემთა ბაზა ხელით თქვენი მონაცემთა ბაზების ადმინისტრირების ინსტრუმენტით (მაგ. phpMyAdmin ) და დაბრუნდით ამ გვერდზე.
